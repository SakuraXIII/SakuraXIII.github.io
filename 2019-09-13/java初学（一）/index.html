类型转换规则

byte , short , char –> int –> long –> float –> double 级别由低到高

由低到高自动转换，由高到低强制转换（因为不同数据类型所占字节位数不同，高位数可以容纳低位数，反之则不行）

同一个表达式中，若存在多种类型，则会被转换为级别最高的类型进行运算
